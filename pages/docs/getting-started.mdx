# Getting started

**Tailwind Variants** combines the power of TailwindCSS with a first-class variant API.

## Installation

To use **Tailwind Variants** in your project, you can install it as a dependency:

<Tabs items={['npm', 'yarn', 'pnpm']}>
  <Tab>
    ```bash copy
    npm install tailwind-variants
    ```
  </Tab>
  <Tab>
    ```bash copy
    yarn add tailwind-variants
    ```
  </Tab>
  <Tab>
    ```bash copy
    pnpm add tailwind-variants
    ```
  </Tab>
</Tabs>

## Usage

<br/>


<TVComponent
  as="button"
  size="sm"
  color="secondary"
  tv={{
    base: 'font-medium bg-blue-500 text-white rounded-full active:opacity-80',
    variants: {
      color: {
        primary: 'bg-blue-500 text-white',
        secondary: 'bg-purple-500 text-white',
      },
      size: {
        sm: 'text-sm',
        md: 'text-base',
        lg: 'px-4 py-3 text-lg',
      },
    },
    defaultVariants: {
      size: 'md',
      color: 'primary',
    },
    compoundVariants: [
      {
        size: ['sm', 'md'],
        class: 'px-3 py-1',
      }
    ]
  }}
>
  Click me
</TVComponent>

```js copy
import { tv } from 'tailwind-variants';

const button = tv({
  base: "font-medium bg-blue-500 text-white rounded-full active:opacity-80",
  variants: {
    color: {
      primary: "bg-blue-500 text-white",
      secondary: "bg-purple-500 text-white",
    },
    size: {
      sm: "text-sm",
      md: "text-base",
      lg: "px-4 py-3 text-lg",
    },
  },
  compoundVariants: [
    {
      size: ["sm", "md"],
      class: "px-3 py-1",
    },
  ],
  defaultVariants: {
    size: "md",
    color: "primary",
  }
});

return (
  <button className={button({ size: 'sm', color: 'secondary' })}>Click me</button>
)
```
