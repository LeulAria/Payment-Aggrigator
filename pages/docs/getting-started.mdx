# Getting started

**Tailwind Variants** combines the power of TailwindCSS with a first-class variant API.

## Installation

To use **Tailwind Variants** in your project, you can install it as a dependency:

<Tabs items={['npm', 'yarn', 'pnpm']}>
  <Tab>
    ```bash copy
    npm install tailwind-variants
    ```
  </Tab>
  <Tab>
    ```bash copy
    yarn add tailwind-variants
    ```
  </Tab>
  <Tab>
    ```bash copy
    pnpm add tailwind-variants
    ```
  </Tab>
</Tabs>

## Usage

<br/>

<TVComponent
  as="button"
  tv={{
    base: 'font-medium bg-blue-500 text-white rounded-full active:opacity-80',
    variants: {
      size: {
        sm: 'px-2 py-1 text-sm',
        md: 'px-3 py-1 text-base',
        lg: 'px-4 py-3 text-lg',
      },
      variant: {
        primary: 'bg-blue-500 text-white',
        secondary: 'bg-gray-500 text-white',
      },
    },
    defaultVariants: {
      size: 'md',
      variant: 'primary',
    }
  }}
>
  Click me
</TVComponent>

```js copy
import { tv } from 'tailwind-variants';

const button = tv('button', {
  base: 'font-medium bg-blue-500 text-white rounded-full active:opacity-80',
  variants: {
    size: {
      sm: 'px-2 py-1 text-sm',
      md: 'px-3 py-1 text-base',
      lg: 'px-4 py-3 text-lg',
    },
    variant: {
      primary: 'bg-blue-500 text-white',
      secondary: 'bg-gray-500 text-white',
    },
  },
  defaultVariants: {
    size: 'md',
    variant: 'primary',
  }
});

<button className={button({ size: 'lg', variant: 'secondary' })}>Click me</button>
```
